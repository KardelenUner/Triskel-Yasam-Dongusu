# 🌀 Anam Cara: Yaşam Döngüsü & Triskel Sarmalı

Mistik Triskelion felsefesinden ilham alınarak geliştirilmiş; bireyin akademik, kişisel ve rutin döngülerini makro ve mikro düzeyde takip etmesini sağlayan bütünsel bir dashboard uygulaması.

 Özellikler
- **Dinamik 3'lü Sarmal Çizimi:** HTML5 Canvas motoruyla görev tamamlanma oranına göre mühürlenen mistik sarmal kolları.
- **Aylık Triskelion Labirenti:** Zaman makinesi gezintisiyle geçmiş aylardaki döngüsel verimlilik takibi.
- **Yıllık Makro Ritmler:** 4x3 matris düzeninde 12 ayın kolektif gelişim ısı haritası.
- **Entelektüel Havuz:** Kitap, film ve dizilerin tarih bazlı mühürlendiği, gelişmiş canlı JS filtreleme mekanizmasına sahip entelektüel arşiv odası.
- **Siber Güvenlik & Mimari:** XSS ataklarına karşı sanitize edilmiş veri akışı, PDO hazırlıklı ifadeler (Prepared Statements) ile SQL Injection koruması.

 Teknolojiler
- **Backend:** PHP 8.x
- **Veri Tabanı:** MySQL
- **Frontend:** HTML5 Canvas, Vanilla JavaScript, CSS3 variables
